✨ NOVO: Paths Dinâmicos

Arquivo criado: app/utils/paths.py

Funções disponíveis:
  get_project_root()         → Raiz do projeto
  get_data_path(file)        → Resolve data/file
  get_config_path(file)      → Resolve config/file
  get_docs_path(file)        → Resolve docs/file
  get_users_csv()            → data/dados_usuarios.csv
  get_sorteios_csv()         → data/dados_loto.csv
  get_tokens_log()           → config/auth_tokens.log

Exemplo de uso:
  from app.utils.paths import get_users_csv
  usuarios_csv = get_users_csv()
  # Retorna: C:\...\data\dados_usuarios.csv
